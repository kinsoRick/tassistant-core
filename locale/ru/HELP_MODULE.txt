`%prefix%помощь` - команда для отображения команд пакетов
`%prefix%почистить` - команда чистит до n сообщений
`%prefix%скачать` - скачивание дополнительные модули команд
`%prefix%мат` - показывает количество произнесенного мата участниками чата
`%prefix%напечатать` - делает анимацию печати для сообщения
`%prefix%процент` - выводит статистику сообщений для каждого пользователя в процентном соотношении
